# Elliot's Bench

A clean static website for a personal legal analysis platform aimed at college students.

## Pages

- `index.html`: Homepage with hero, mission framing, and featured analysis.
- `posts.html`: Blog-style posts index.
- `current-events.html`: Section for recent posts.
- `past-analyses.html`: Archive landing page with category placeholders.
- `timeline.html`: Placeholder for a future interactive timeline.
- `post-campus-speech.html`: Sample post using the Issue, Law, Side A, Side B, Analysis structure.
- `post-presidential-immunity.html`: Trial Bench Brief with sources.
- `post-chevron.html`: Trial Bench Brief with sources.
- `post-affirmative-action.html`: Trial Bench Brief with sources.
- `post-dobbs.html`: Bench Brief with sources.
- `post-bruen.html`: Bench Brief with sources.
- `post-west-virginia-epa.html`: Bench Brief with sources.
- `why.html`: Mission page.
- `questions.html`: Contact/questions prototype.

## How to view

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Future app path

This static version can later become a React, Next.js, or mobile app project. The repeated post structure is intentionally simple so posts can eventually be stored as data and generated from a form, CMS, or AI-assisted workflow.
